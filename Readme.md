# AWS + LLM Study
---
## Install virtualenv
```
pip install virtualenv
virtualenv venv
```

## Activate env
```
venv\Scripts\activate.bat        (Windows)
source venv\Scripts\activate     (Linux , MacOS)
```

## Install Project
```
pip install -r requirements.txt
```

## Set OpenAI Key
- Create '.env' file and write key
```
OPENAI_API_KEY='KEY'
```
